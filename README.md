
# Shinigami World

The official anime version of "[The Lost Ball](https://github.com/Gamesflow/The-Lost-Ball)"

## How to play 

|Key| Controls |
|--|--|
| Spacebar/Up arrow | Jump |
| Right Arrow | Go Right |
| Left Arrow | Go left |
